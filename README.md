# Agent Command Hub

A real-time mission control dashboard for AI agents. Built with React, TypeScript, Supabase, and Tailwind CSS.

Give any AI agent — Claude Code, OpenClaw, Python scripts, or custom — a visual command center where their work becomes visible, trackable, and autonomous.

## What It Does

- **Kanban Task Board** — 5-column board (To Do, Doing, Needs Input, Canceled, Done) with drag-and-drop
- **AI Activity Log** — Categorized journal of everything your agents do
- **Agent Status Ring** — Real-time presence indicator (green/yellow/red/gray)
- **Questions & Approvals** — Async Q&A between you and your agents
- **HTML Reports** — Formatted reports generated by agents on task completion
- **Scheduled Automations** — Cron-based jobs (morning digest, evening report, competitor intel)
- **Animated 2D Office** — AI agents as animated characters in a virtual workspace
- **OpsCenter** — Modular app system with 19+ block types for custom dashboards
- **Multi-Agent System** — Multiple agents with distinct identities, skills, and memory
- **Cognitive Memory** — Persistent knowledge base that survives across sessions
- **Skill Factory** — Configurable API skills with version control

## Quick Start

```sh
git clone https://github.com/polymathproject1-rgb/agent-command-hub.git
cd agent-command-hub
npm i
```

### Environment Setup

```sh
cp .env.local.example .env.local
cp .env.server.example .env
```

Set values in `.env` (server-side):

```env
AGENT_COMMAND_API_URL=https://<your-supabase-project>.supabase.co/functions/v1/ai-tasks
AGENT_COMMAND_WEBHOOK_SECRET=your-secret-here
```

Set values in `.env.local` (frontend):

```env
VITE_AGENT_NAME=YourAgent
VITE_AGENT_EMOJI=🤖
```

### Run

```sh
npm run dev
```

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend:** Supabase (PostgreSQL + Edge Functions + Realtime)
- **Charts:** Recharts
- **Animations:** Framer Motion
- **Drag & Drop:** @dnd-kit

## Agent Integration

Any agent that can make HTTP POST requests can connect. The API gateway is a single Supabase Edge Function (`ai-tasks`) that routes by `request_type` + `action`.

Compatible with:
- **Claude Code** — Drop `CLAUDE.md` in your project
- **OpenClaw** — REST API
- **Python / Node.js** — Standard HTTP requests
- **Any language** — Just POST JSON with your webhook secret
